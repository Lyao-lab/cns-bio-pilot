# Reporting Guidelines (EQUATOR Network)

Use the matching reporting guideline for your study design. Many journals require the completed checklist as a supplement with page/line numbers, plus an adherence statement in Methods.

## By Study Design (Quick Match)

| Study type | Guideline | Version |
|---|---|---|
| Randomized controlled trial | **CONSORT** | 2010 |
| Cohort / case-control / cross-sectional | **STROBE** | 2007 |
| Systematic review / meta-analysis | **PRISMA** | 2020 |
| Trial protocol | **SPIRIT** | 2013 |
| Diagnostic accuracy | **STARD** | 2015 |
| Prediction model (dev / validation) | **TRIPOD** | 2015 |
| Animal research | **ARRIVE** | 2.0 (2020) |
| Case report | **CARE** | 2013 |
| Quality improvement | **SQUIRE** | 2.0 |
| Economic evaluation | **CHEERS** | 2022 |
| Qualitative research | **SRQR** / **COREQ** | 2014 |

## CONSORT (RCTs) — Key Checklist Items

- Title identifies study as randomized trial; structured abstract
- Trial registration number
- **Eligibility criteria** + settings/locations
- Interventions described for replication
- Primary/secondary outcomes with definition
- Sample size determination (power calc)
- **Randomization**: sequence generation, allocation concealment, implementation
- **Blinding**: who was blinded (participants, providers, outcome assessors)
- Statistical methods for primary/secondary, interim analyses
- Participant **flow diagram**: enrollment → allocation → follow-up → analysis, with reasons for loss
- Baseline characteristics table; harms/adverse events
- Limitations, generalizability, interpretation
- Funding source + role

Extensions exist for cluster, non-inferiority, pragmatic, crossover, N-of-1 trials.

## STROBE (Observational) — Key Items

- Title/abstract indicate design (cohort / case-control / cross-sectional)
- Variables defined: outcomes, exposures, confounders
- Data sources + measurement methods
- Bias-mitigation strategies
- Study size justification
- Statistical methods: confounding adjustment, interactions, sensitivity analyses, subgroups
- Participant flow with reasons for non-participation
- Results: unadjusted + adjusted estimates with CI
- Limitations: discuss sources of bias, confounding

Extensions: STROBE-ME (molecular epi), RECORD (routinely-collected data).

## PRISMA 2020 (Systematic Reviews) — Key Items

- Registration (PROSPERO); protocol availability
- Eligibility criteria
- Information sources: databases, registers, dates
- **Full search strategy** for ≥1 database
- Selection process (# screeners, dispute resolution)
- Data items extracted
- **Risk of bias**: tool used (ROB 2 / ROBINS-I / QUADAS-2)
- Effect measures; synthesis method (meta-analysis settings if any)
- **Flow diagram**: identification → screening → inclusion; separate database vs. register; reports vs. studies
- Certainty of evidence: GRADE

Extensions: PRISMA-P (protocols), PRISMA-ScR (scoping), PRISMA-NMA (network meta-analysis).

## SPIRIT (Trial Protocols)

33 items covering: administrative info (registration, funding), background/rationale/objectives, trial design, eligibility, intervention, outcomes, timeline, sample size, recruitment, allocation/randomization/blinding, data collection/management, statistical methods, monitoring (DSMB), harms, ethics, dissemination.

## STARD (Diagnostic Accuracy) — Key Items

- Index test + reference standard description; rationale for reference standard
- Participant selection + recruitment (consecutive? random?)
- Test cut-offs; handling of indeterminate results
- Participant flow + timing between tests
- Cross-tabulation of index vs. reference
- Accuracy estimates with CI (sensitivity, specificity, +/- LR, AUC)
- Adverse events from testing

## TRIPOD (Prediction Models) — Key Items

- Title/abstract identify as prediction model study
- Type: development only, development + validation, external validation, updating
- Data source + participants; outcome definition
- Candidate predictors; sample size justification
- Missing-data handling
- Model-building procedure; model specification (full equation/algorithm)
- Performance: discrimination (C-statistic/AUC) + calibration
- External validation results if applicable

## ARRIVE 2.0 (Animal Research)

**Essential 10** (minimum requirement):
1. Study design
2. Sample size calculation
3. Inclusion/exclusion criteria
4. Randomization
5. Blinding
6. Outcome measures (primary/secondary)
7. Statistical methods
8. Experimental animals (species, strain, sex, age, weight, source)
9. Experimental procedures (detail for replication)
10. Results + interpretation

**Recommended set** (additional): ethical statement, housing/husbandry, animal care/monitoring, interpretation, protocol registration, data access.

## CARE (Case Reports) — Key Items

Title with "case report"; abstract; patient demographics + main concerns; clinical findings; **timeline of events**; diagnostic assessment; therapeutic intervention; follow-up + outcomes; patient perspective; informed consent.

## Adherence Statement (Methods)

```
This study was reported in accordance with the [GUIDELINE] statement ([version/year]). The completed checklist is provided as Supplementary File [N].
```

## Pre-Submission Checklist

- [ ] Correct guideline selected for design (consult EQUATOR if unclear)
- [ ] Latest version used
- [ ] Every checklist item addressed with page/line location
- [ ] Flow diagram included where required (CONSORT, PRISMA, STARD)
- [ ] Protocol registration cited (PROSPERO, ClinicalTrials.gov)
- [ ] Adherence statement in Methods + cover letter
- [ ] Completed checklist uploaded as supplement if journal requires
